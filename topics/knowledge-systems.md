# Knowledge Systems

Context engineering, skill graphs, second brains, memory architectures, and how to structure knowledge for humans and agents.

## Key Themes

### Memory as Infrastructure
Agent memory files and decision logs create institutional knowledge that outlives individuals. This is context engineering made operational.

### Knowledge That Compounds
Compound engineering, spec-first development, and cross-user knowledge transfer all share a pattern: structured capture of what was learned during work, not after it.

### Progressive Disclosure Over Flat Search
Complex domains need interconnected knowledge that agents navigate progressively — index → description → links → sections → full content. Skill graphs, knowledge graphs, and PageIndex all use this pattern.
